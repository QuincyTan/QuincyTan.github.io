湘潭大学2019程序设计实践

发表于 2019-06-03   |   更新于 2019-06-06 | 分类于 ACM | 阅读次数:
抄袭一时爽，查重火葬场
因考前会查重，故仅提供思路参考，代码后续补上，有更好解法欢迎交流
1262 Fish

贪心，优先队列维护每个池塘能钓到鱼的数量即可，当前池塘有最多鱼的池塘优先出队列。时间复杂度O(Tnlogn+Tmnlogn)

1271 Color

结论：n*m-(x的种类数)*(y的种类数)

1163 ASCLL

直接模拟，输出%05X和%02X即带前导0的十六进制，回车的ASCLL码也要输出，注意输出空格和回车的地方

1243 Bob’s Password

枚举九个数字会非法的情况即可

1246 Matrix Transposition

看半天题意，n,m没用，将输入的x看做y，输入的y看做x，反过来按y优先排序即可，时间复杂度O
（T*KlogK）

1248 Alice and Bob

直接模拟，将1作为数字7处理会很好写很多，注意求和依然是+1

1251 Colombian Number

简单数学，注意到[1,1000000000]区间内数位和在[1,81]内，所以只需在区间[n-81,n]内枚举k看是否能使k+sum(k)==n成立即可

1253 Robot

贪心，如果在中途转弯肯定不如每次从头走到尾，可以证明最多转n-1次，时间复杂度O($n^2$)

超时的试试交C ，C++考虑不用万能头，还不行加快读 /手动滑稽

1252 Matrix Word

统计每个字符在每一行每一列的出现次数，最后判断是否都为1即可

1621 Duplicate Elements

unordered_map或map或set记录每个值是否出现即可,当然也可以用其他的哈希方法，超时的可以试试之前的方法，时间复杂度O(n)或O(nlogn)

1263 矩形面积的并

哪一年的蓝桥杯原题来着，求出相交矩形的边界,判掉不会相交的情况，在不知道位置相对位置的情况下计算面积注意取绝对值，面积并=两矩形面积和-两矩形面积交

l=max(min(x1,x2),min(x3,x4));
r=min(max(x1,x2),max(x3,x4));
d=max(min(y1,y2),min(y3,y4));
u=min(max(y1,y2),max(y3,y4));

1268 鞍点

输入的时候就记录每行每列的最值即可

1272 Robot

统计L,R,U,D的个数，只有当其中三个有且三个的其中一个变为剩下两个中另一个使得剩下两个相等时才可行

1277 Tri-Triangle

找规律然后直接输出

1280 String Hash

就是26进制转10进制

1281 Cute String

输入用gets或者循环getchar()，跳过字符串首部空格后，统计中间空格个数便可知道单词个数，看空格个数是否>9和字母个数>10即可
